# NAVBATCHI AI - Stomatologiya Klinikai Navbat Tizimi

🏥 **NAVBATCHI AI** - Zamonaviy va innovativ stomatologiya klinikasining web-ilovasi.

## ✨ Xususiyatlari

✅ **3 tildagi qo'llab-quvvatlash** (Uzbek, Rus, Ingliz)  
✅ **Oson navbat olish tizimi** - 4-qadam forma  
✅ **Shifokorlar profili** - 6 ta tajribali shifokor  
✅ **Admin panel** - Navbatlarni va shifokorlarni boshqarish  
✅ **Responsive dizayn** - Barcha qurilmaga mos  
✅ **Modern UI/UX** - Qizil-Oq-Qora rang sxemasi  
✅ **To'lov usullari** - Naqd, Karta, Payme, Click  
✅ **Shifokorlar statistikasi** - Admin panelidagi ma'lumotlar  

## 🚀 Boshlash

### O'rnatish

```bash
# Git repositoriyani klonlash
git clone https://github.com/azimjonashuraliyev80-ux/Navbatchi-ai.git
cd Navbatchi-ai

# Kerakli paketlarni o'rnatish
npm install

# Developement serverini boshlash
npm start
```

## 📄 Sahifalar

### 🏠 Home (Bosh Sahifa)
- Klinika haqida qisqa ma'lumot
- Xizmatlar to'plami
- Nima uchun biz?
- CTA (Call to Action) tugmalar

### 📅 Appointment (Navbat Olish)
- 4-qadam registratsiya formasi
- Shifokor va xizmatni tanlash
- Vaqt slotlarini tanlash
- To'lov usulini tanlash
- Mavjud navbatlarning jadvalini ko'rish

### 👨‍⚕️ Doctors (Shifokorlar)
- 6 ta shifokorning profili
- Mutaxassisligi va tajribasi
- Sertifikatlari
- Ta'limi haqida ma'lumot

### ℹ️ About (Haqimizda)
- Klinika tarixi
- Missiyon va vizyon
- Qadriyatlar
- Statistika
- Kontakt ma'lumotlari

### ⚙️ Admin Panel
- Navbatlarni boshqarish
- Shifokorlarni boshqarish
- Statistika va hisobotlar
- Qidirish va filtrlash
